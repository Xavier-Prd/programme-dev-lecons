Les méthodes en Ruby

Thème : Ruby fondamentaux — Sous-thème : méthodes · Difficulté : débutant · ~35 min
01
Première méthode — saluer
⏱ ~5 min
Crée une méthode saluer qui prend un prénom en paramètre et retourne la chaîne "Bonjour, [prénom] !". Teste-la avec plusieurs prénoms.
CONTRAINTES
  • Utilise le mot-clé def et end
  • Utilise l'interpolation de chaîne #{...}
  • Appelle la méthode au moins 3 fois avec des prénoms différents
  • Affiche le résultat avec puts
▶ VOIR LA CORRECTION 
def saluer(prenom)
  "Bonjour, #{prenom} !"
end

puts saluer("Xavier")
puts saluer("Clément")
puts saluer("Alice")
💡 En Ruby, la dernière expression évaluée dans une méthode est automatiquement retournée. Le mot-clé return est optionnel (mais utilisable).
02
Paramètres par défaut
⏱ ~8 min
Crée une méthode presenter qui prend deux paramètres : nom et role (avec "développeur" comme valeur par défaut). Elle retourne "Je m'appelle [nom] et je suis [role].".

Appelle-la une fois avec les deux paramètres, et une fois sans fournir le rôle.
CONTRAINTES
  • Utilise la syntaxe def presenter(nom, role = "...")
  • Teste les deux cas d'appel
  • Affiche les résultats
▶ VOIR LA CORRECTION 
def presenter(nom, role = "développeur")
  "Je m'appelle #{nom} et je suis #{role}."
end

puts presenter("Xavier", "ingénieur")
puts presenter("Clément")
# => "Je m'appelle Clément et je suis développeur."
03
Méthode avec calcul — IMC
⏱ ~10 min
Crée une méthode calculer_imc qui prend poids (en kg) et taille (en mètres) et retourne l'IMC arrondi à 1 décimale.

Crée ensuite une méthode categorie_imc qui prend un IMC et retourne la catégorie : "Insuffisance pondérale" (< 18.5), "Poids normal" (18.5–24.9), "Surpoids" (25–29.9), "Obésité" (>= 30).
CONTRAINTES
  • Formule IMC : poids / (taille * taille)
  • Utilise .round(1) pour arrondir
  • Utilise if / elsif / else dans la méthode catégorie
  • Affiche : "IMC : 22.5 → Poids normal"
▶ VOIR LA CORRECTION 
def calculer_imc(poids, taille)
  (poids / (taille * taille)).round(1)
end

def categorie_imc(imc)
  if imc < 18.5
    "Insuffisance pondérale"
  elsif imc < 25
    "Poids normal"
  elsif imc < 30
    "Surpoids"
  else
    "Obésité"
  end
end

imc = calculer_imc(70, 1.75)
puts "IMC : #{imc} → #{categorie_imc(imc)}"
# => IMC : 22.9 → Poids normal
04
Méthode récursive — compte à rebours
⏱ ~8 min · plus difficile
Crée une méthode compte_a_rebours qui prend un entier n et affiche les nombres de n jusqu'à 0, puis affiche "🚀 Décollage !".

Implémente-la d'abord avec une boucle while, puis en version récursive (la méthode s'appelle elle-même).
CONTRAINTES
  • Version 1 : utilise while n >= 0
  • Version 2 : la méthode doit s'appeler elle-même avec n - 1
  • La version récursive doit avoir un cas de base (quand n < 0, arrêter)
  • Résultat attendu : 5, 4, 3, 2, 1, 0, 🚀 Décollage !
▶ VOIR LA CORRECTION 
# Version boucle
def compte_a_rebours_boucle(n)
  while n >= 0
    puts n
    n -= 1
  end
  puts "🚀 Décollage !"
end

# Version récursive
def compte_a_rebours(n)
  return puts "🚀 Décollage !" if n < 0
  puts n
  compte_a_rebours(n - 1)
end

compte_a_rebours(5)
💡 La récursion, c'est une méthode qui s'appelle elle-même jusqu'à atteindre un cas de base. Sans cas de base → boucle infinie → erreur SystemStackError.
