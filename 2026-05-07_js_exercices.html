JavaScript Moderne

Thème : const/let · arrow functions · destructuring · spread · modules ES6 · ~75 min

◈ Introduction (20%)

JavaScript a beaucoup évolué depuis ES6 (2015). On abandonne var pour deux nouveaux mots-clés : const (valeur qui ne changera pas) et let (valeur qui peut changer). Cette distinction rend le code plus prévisible.

Les arrow functions sont une syntaxe compacte pour écrire des fonctions :

// Fonction classique
function double(n) { return n * 2; }

// Arrow function équivalente
const double = (n) => n * 2;

Le destructuring permet d'extraire des valeurs d'un tableau ou d'un objet en une ligne :

const [a, b] = [10, 20];              // tableau
const { nom, age } = { nom: "Alice", age: 30 }; // objet

L'opérateur spread (...) étale les éléments d'un tableau ou d'un objet :

const arr1 = [1, 2, 3];
const arr2 = [...arr1, 4, 5];  // [1, 2, 3, 4, 5]

Les modules ES6 permettent d'organiser le code en fichiers séparés avec export et import. Tu les verras en action dans les exercices avancés.

Mini-projet — traitement d'une liste de produits
⏱ ~20 min · exercice intégrateur
