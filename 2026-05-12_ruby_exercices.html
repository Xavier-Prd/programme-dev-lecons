EX 01

Définir ses premières méthodes

facile

En Ruby, une méthode se définit avec def. Elle retourne automatiquement la valeur de la dernière expression évaluée.

Définis les trois méthodes suivantes :

  • saluer(prenom) → retourne "Bonjour, [prenom] !"
  • aire_rectangle(largeur, hauteur) → retourne l'aire (largeur × hauteur)
  • carre(n) → retourne n²

CONTRAINTES

— N'utilise pas return explicite (laisse Ruby retourner la dernière expression)
— Teste chaque méthode avec au moins deux appels puts

⏱ ~8 min
▶ Voir la correction 
def saluer(prenom)
  "Bonjour, #{prenom} !"
end

def aire_rectangle(largeur, hauteur)
  largeur * hauteur
end

def carre(n)
  n * n
end

puts saluer("Alice")        # Bonjour, Alice !
puts saluer("Bob")          # Bonjour, Bob !
puts aire_rectangle(4, 5)   # 20
puts aire_rectangle(10, 3)  # 30
puts carre(7)               # 49
puts carre(12)              # 144
EX 02

Paramètres par défaut

facile

Les paramètres par défaut permettent d'appeler une méthode sans fournir tous les arguments.

Définis la méthode message_bienvenue(nom, langue = "fr") qui :

  • Si langue == "fr" → retourne "Bienvenue, [nom] !"
  • Si langue == "en" → retourne "Welcome, [nom]!"
  • Sinon → retourne "Hello, [nom]."

CONTRAINTES

— Utilise if / elsif / else
— Appelle la méthode sans le 2e argument, puis avec "en", puis avec une autre valeur

⏱ ~8 min
▶ Voir la correction 
def message_bienvenue(nom, langue = "fr")
  if langue == "fr"
    "Bienvenue, #{nom} !"
  elsif langue == "en"
    "Welcome, #{nom}!"
  else
    "Hello, #{nom}."
  end
end

puts message_bienvenue("Marie")         # Bienvenue, Marie !
puts message_bienvenue("John", "en")    # Welcome, John!
puts message_bienvenue("Yuki", "jp")    # Hello, Yuki.
EX 03

Méthode avec plusieurs valeurs de retour

moyen

Ruby permet de retourner plusieurs valeurs d'une méthode sous forme de tableau implicite.

Définis la méthode stats(nombres) qui reçoit un tableau d'entiers et retourne un tableau [min, max, somme, moyenne].

CONTRAINTES

— Utilise les méthodes Ruby .min, .max, .sum
— Calcule la moyenne en divisant par le nombre d'éléments
— Destructure le résultat avec min, max, somme, moy = stats([...])

# Exemple attendu :
min, max, somme, moy = stats([4, 2, 9, 1, 7])
puts min    # 1
puts max    # 9
puts somme  # 23
puts moy    # 4.6
⏱ ~8 min
▶ Voir la correction 
def stats(nombres)
  min    = nombres.min
  max    = nombres.max
  somme  = nombres.sum
  moyenne = somme.to_f / nombres.length
  [min, max, somme, moyenne]
end

min, max, somme, moy = stats([4, 2, 9, 1, 7])
puts min    # 1
puts max    # 9
puts somme  # 23
puts moy    # 4.6

min2, max2, _, _ = stats([10, 5, 8])
puts "#{min2} / #{max2}"  # 5 / 10
EX 04

Méthode récursive — Factorielle

moyen

Une méthode peut s'appeler elle-même : c'est la récursion. Chaque appel traite un cas plus petit jusqu'à atteindre un cas de base.

Définis factorielle(n) de façon récursive :

  • Si n == 0 → retourne 1 (cas de base)
  • Sinon → retourne n * factorielle(n - 1)

CONTRAINTES

— N'utilise pas de boucle
— Teste avec 0, 1, 5, 10
— Ajoute une garde : si n < 0, lève une exception ArgumentError

# Résultats attendus :
# factorielle(0) → 1
# factorielle(5) → 120
# factorielle(10) → 3628800
⏱ ~10 min
▶ Voir la correction 
def factorielle(n)
  raise ArgumentError, "n doit être >= 0" if n < 0
  return 1 if n == 0
  n * factorielle(n - 1)
end

puts factorielle(0)   # 1
puts factorielle(1)   # 1
puts factorielle(5)   # 120
puts factorielle(10)  # 3628800

begin
  factorielle(-3)
rescue ArgumentError => e
  puts "Erreur : #{e.message}"
end
EX 05

Mini-projet — Calculatrice Ruby

challenge

Combine tout ce que tu as vu pour construire une mini-calculatrice.

Définis les méthodes suivantes :

  • additionner(a, b)
  • soustraire(a, b)
  • multiplier(a, b)
  • diviser(a, b) → retourne nil et affiche un message si b == 0
  • calculer(a, operateur, b) → délègue à la bonne méthode selon operateur ("+", "-", "*", "/"). Si l'opérateur est inconnu, retourne "Opérateur inconnu"

CONTRAINTES

— calculer utilise case/when ou des if/elsif
— Teste 6 appels différents à calculer, dont une division par zéro et un opérateur inconnu

⏱ ~12 min
▶ Voir la correction 
def additionner(a, b) = a + b
def soustraire(a, b)  = a - b
def multiplier(a, b)  = a * b

def diviser(a, b)
  if b == 0
    puts "Erreur : division par zéro"
    nil
  else
    a.to_f / b
  end
end

def calculer(a, operateur, b)
  case operateur
  when "+" then additionner(a, b)
  when "-" then soustraire(a, b)
  when "*" then multiplier(a, b)
  when "/" then diviser(a, b)
  else "Opérateur inconnu"
  end
end

puts calculer(10, "+", 5)   # 15
puts calculer(10, "-", 3)   # 7
puts calculer(4, "*", 7)    # 28
puts calculer(9, "/", 3)    # 3.0
puts calculer(5, "/", 0).inspect  # nil (+ message erreur)
puts calculer(5, "%", 2)    # Opérateur inconnu
